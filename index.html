Projects

Capital

React Redux SASS ES6 webpack 2

Capital is a HTML5 game built using jQuery, SASS (SCSS) and RequireJS. This project is a proof of concept by using Scalable Vector Graphics (SVG) for rendering.

Each frame in the game is scalable and zoomable without any loss of quality. It is currently being re-written as a React+Redux app using ES6.

InHere

Android Java PHP MySQL

InHere is a location-based information advertising system. The system is a combination of CMS, web service and a demo mobile app built using Android SDK.

The system uses Bluetooth low energy device for geo-fencing and Bluetooth for data delivery. When user enters a geo-fenced area, the Android app will recognize the user's current location and serve information according to the location.
